Setup

In order to access Rocket League, it's important to have the game present either on Steam or on Epic Games since we will be using it on PC. The instructions go as follows:

  1. Download one of the launchers

    2. As of now, the game is no longer on Steam but can be downloaded for free on Epic Games. But you'll need to download the launcher first: Download Epic Games here

  2. Create an account or Sign In

    3. You can use the link given above to create your account while downloading the launcher.

  3. Install the game

    4. On the Launcher, you can easily download the game by following these steps:

    1. On the Store page, search for Rocket League
    2. Select "Add to Library"
    3. In your Library, begin download by clicking on the game
  4. Boot the title

    5. You're now ready to play!
